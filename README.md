# Stocks

Best socks store ever!!!!

##Scrapper:

`cd products_scraper`

`pip3 install -r requirements.txt`

`python3 scrapper.py`

##Server:

`cd server`

`npm install`

`npm run start-server`

##Admin:

`cd admin`

`npm install`

`npm install @angular/cli`

`ng serve`

##Ecom:

`cd stocks-ecom`

`npm install`

`npm run dev`
